XU4 is a remake of the computer game Ultima IV. The goal is to make it easy and convenient to play this classic on modern operating systems. XU4 is primarily inspired by the much more ambitious project Exult. XU4 is a cross-platform application available for Linux and Windows.

The game is fully playable from beginning to end. The latest source code can always be accessed from both SourceForge and GitHub.

A secondary goal of this project is to clearly document the formats of the data files from the original Ultima IV. This work in progress is available here: FileFormats.md

Please see the README file for information on running xu4. To see a list of recent improvements have a look at the Change Log.

The original Ultima IV is freeware. Download it here!
