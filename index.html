Important News:

Version 1.1 is available for Linux and Windows!

This release uses a new audio backend and some missing sound effects were added. Separate soundtrack modules can now be loaded from inside the game. The ChangeLog contains the full list of improvements.

Note that while graphic modifications are technically possible, alternate tilesets will not be officially supported until a later version.

XU4 is a remake of the computer game Ultima IV. The goal is to make it easy and convenient to play this classic on modern operating systems. XU4 is primarily inspired by the much more ambitious project Exult. XU4 is a cross-platform application available for Linux and Windows. Porting to other platforms with Allegro 5 support should be simple, but the project needs developers who use them.

The game is fully playable from beginning to end. The latest source code can always be accessed from both SourceForge and GitHub.

A secondary goal of this project is to clearly document the formats of the data files from the original Ultima IV. This work in progress is available here: FileFormats.md

Please see the README file for information on compiling, installing and running xu4.

The original Ultima IV is freeware. Download it here!
